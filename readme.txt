Adds a sidebar widget that display a low carbon cooling calculator.


== Description ==

This plugin adds a sidebar widget that displays a low carbon cooling calculator, you input the details of the location that you require cooling for and the calculator recommends the approptiate low carbon cooling unit(s).

= Version 1.0 =
First release.


== Installation ==

1. Unpack the *.zip file and extract the /carbon-calc/ folder and the files.
2. Using an FTP program, upload the full /carbon-calc/ folder to your WordPress plugins directory (Example: /wp-content/plugins).
3. Go to Plugins > Installed and activate the plugin.
4. Go to Appearance > Widgets, add the widget to a sidebar and save.


== Frequently Asked Questions ==

= The plugin is great but can I change the colour to fix in better with my WordPress theme? =
Yes you can. Simply change the background image `carbon-calc/images/carbon-calc.jpg` and the colour of `container` in `carbon-calc/styles/carbon-calc.css`.
